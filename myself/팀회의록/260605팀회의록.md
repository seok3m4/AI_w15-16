# 10. T자형(T-Shaped) 프로젝트 논의

코치가 언급한 내용

---

## 기능 확장형

가로 방향

```text
기능 많음
```

예시

```text
카카오톡

채팅

친구

프로필

송금

검색
```

---

## 깊이 확장형

세로 방향

```text
기능은 적지만

하나를 깊게 구현
```

예시

```text
메신저

실시간 통신

읽음 처리

대용량 메시지

알림 시스템
```

---

### 팀이 해야 할 일

서비스 후보 여러 개 생성

예시

```text
후보 A

후보 B

후보 C
```

---

이후

```text
기술 난이도

흥미

확장성
```
